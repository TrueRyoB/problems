# Rhythm Co, Ltd.
There are $N$ instruments.

Machine $i$ costs $C_i$, and plays a value $A_i$ at time modulo $M_i$.

You may purchase one or more of each machine.

You are given $Q$ queries. 

For each query, determine the maximum total values played till time $T_i$ with a budget $B_i$.

## Constraints
- $1 \leq N \leq 10^5$
- $1 \leq C_i \leq 10^{10}$
- $1 \leq A_i \leq 10^{10}$
- $1 \leq M_i \leq 10^{10}$
- $1 \leq Q \leq 10^5$
- $0 \leq T_i \leq 10^{10}$
- $1 \leq B_i \leq 10^{10}$
