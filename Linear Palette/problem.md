## Linear Palette

There is an indefinitely long linear palette.

You are given $Q$ queries.

For query type 1, dye every discrete cell in range $[l, r]$ by at least color $z$.

For query type 2, undye every discrete cell in range $[l, r]$ at color less or equal to $z$.

For query type 3, solve the total covered area in range $[l, r]$ by any colors but $z$.

### Constrainsts
- $1 \leq Q \leq 10^5$
- $-10^9 \leq l \leq r \leq 10^9$
- $1 \leq z \leq 10^5$
